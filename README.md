# Danu — Gestión de proyectos para equipos

Aplicación SaaS de gestión de proyectos con Kanban en tiempo real, Team Health Score y arquitectura en 5 capas.

## Stack

Next.js 16 · React 19 · TypeScript · Tailwind CSS · Prisma · Supabase · Auth.js v5

## Correr localmente

```bash
git clone https://github.com/tu-usuario/danu
cd danu
npm install
cp .env.example .env.local   # agregar tus claves
npx prisma db push
npm run dev
```
